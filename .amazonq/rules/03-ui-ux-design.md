# UI/UX Design Rules

## Dark Mode Only

- Focus exclusively on dark variant
- Light mode is NOT supported

## Minimalist Black Styling

- **Primary colors**: Black, white, and gray tones
- **No gradients** unless explicitly requested
- **Subtle hover effects** using opacity and color transitions
- **Clean, sharp edges** with minimal border-radius

## Color Palette

- Background: `#0a0e27` (dark navy)
- Primary: Black (`#000000`)
- Text: White (`#ffffff`)
- Muted: Gray tones (`gray-300`, `gray-400`, `gray-500`)
- Accents: Use sparingly, primarily white

## Responsive Design

- Mobile-first approach
- Test on all breakpoints (mobile, tablet, desktop)
- Use Tailwind responsive utilities

## Accessibility

- Use semantic HTML elements
- Add ARIA labels where needed
- Ensure keyboard navigation support
- Maintain sufficient color contrast
- Test with screen readers
